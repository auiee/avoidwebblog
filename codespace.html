github codespaceみたいなサービス

  1. github codespaceのようなサービス github codespacesは、一ヶ月に120時間の時間制限があります。 なのでずっと動かすことには向きません。 そこで今回はほかのcodespaceのサービスを紹介します。
  2. 1 codeanywhere codeanywhereは、githbのcodespaceの機能だけ取り出したようなサービスです このサービスはgithubアカウントがあれば無料で登録できます。 おそらくgithubのような時間制限もありません。
  3. 2 gitpod このサービスも同じくcodespaceのサービスです。 こちらもgithubアカウントで登録できます。 こちらはサーバーの性能を2つから無料で選べます。 こちらもおそらく制限時間はありません。
  4. 3 replit こちらはcodespace単体というより、他の人の作品も見れたりします。 同じくgithubアカウントで登録できますが、このサービスは3つまでしかrepl (プロジェクトみたいなもの) が作れないのであまり推奨しませんが、runボタン一つで動かせるのは便利です。
